## Concerns
- **No CI/CD**: No GitHub Actions, no automated testing on push, no build verification
- **No automated linting in CI**: The `lint` script exists but no enforcement mechanism
- **Loose TypeScript**: `strictNullChecks: false`, `noImplicitAny: false` means many potential null reference bugs are hidden
- **No test files found**: Despite vitest being configured, no `*.test.ts` or `*.spec.ts` files were observed in the directory listing
- **Dev proxy to production**: The dev server proxies to `list-service.icustomer.ai` and `prod-aloop-api-gateway.icustomer.ai`, meaning local development hits real production services
- **`@typescript-eslint/no-unused-vars` disabled**: Dead code accumulates silently
- **No pre-commit hooks**: No husky, lint-staged, or similar tools configured
