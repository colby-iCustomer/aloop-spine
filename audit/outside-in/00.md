# 00 - Product Context

## What Docs Say

Audience Loop (ALoop) is a **PLG SaaS product** built by iCustomer. It is a multi-tenant data enrichment platform that helps B2B marketing and sales teams build, enrich, and export audience lists to advertising and outreach platforms.

### Core Value Proposition
Users upload or create contact/company lists, then enrich them using AI-powered data agents. Enriched data can be exported to ad platforms (LinkedIn Ads, Meta Ads, Google Ads, Reddit Ads), outreach tools (SmartLeads, Mailshake), CRMs (HubSpot, Salesforce), and spreadsheets (Google Sheets, CSV).

### User Types
- **Full Users** - business email domain, own workspace created on signup, roles: Owner/Admin/Editor/Viewer
- **Guest Viewers** - personal email domain (gmail, yahoo, etc.), read-only access, receive a dormant personal workspace, can only be invited on Pro plan

### Key Product Concepts
1. **Templates** - define column structure and enrichment capabilities for lists. Templates are activation overlays, not standalone entities. A list can have multiple template activations. Templates are organized by channel (LinkedIn, Meta, Google Ads), source (prospecting, enrichment), and cross-channel workflows.
2. **Workspaces** - top-level isolation boundary. Every resource (folder, list) belongs to exactly one workspace. Users access workspaces through membership.
3. **Folders (Projects)** - organizational containers for lists within a workspace.
4. **Lists (Datasets)** - the core data unit. Rows of contact/company data in a spreadsheet-like AG Grid interface.
5. **Enrichment** - AI-powered process that fills empty cells using external data providers. Credit-based pricing.
6. **Credits** - consumption currency for enrichment operations. Three plan tiers: Free ($0, 200 credits), Starter ($99, 2000 credits), Professional ($299, 7000 credits).
7. **Live Audiences** - uploaded audience segments on ad platforms (LinkedIn, Meta) with campaign attachment capabilities.

### Who Uses It
B2B marketers and sales teams who need to:
- Build targeted prospect lists
- Enrich contact data (emails, phones, company info, social profiles)
- Export enriched audiences to advertising platforms
- Track audience performance across platforms

### Product Architecture
The app follows a template-driven workflow:
1. User picks a template from the gallery (e.g., LinkedIn Prospecting, Contact Enrichment)
2. Template maps fields to list columns via AI field matching
3. User enriches data (credits consumed per row)
4. User exports to destination platform

The product also supports MCP (Model Context Protocol) access, meaning the API should be self-describing enough for programmatic callers, not just the frontend.
