# 05 - API Contracts

## What Docs Say

### Service Topology

| Service | Base URL | Vite Proxy | Handles |
|---------|----------|------------|---------|
| list-service | `https://list-service.icustomer.ai/api/v1` | `/api/v1` | Projects, lists, data, columns, enrichment, payments, auth |
| gateway | `https://prod-aloop-api-gateway.icustomer.ai` | `/gateway` | Integration OAuth, exports, analytics, prospecting |

### Response Convention
All responses: `{ success: true, data: T, pagination?: {...} }` or `{ success: false, error: string }`

### API Clients
- `apiClient` - fetch-based singleton for list-service
- `gatewayClient` - fetch-based for gateway
- Token refresh on 401 + `TOKEN_EXPIRED` code with automatic retry
- `skipAuth: true` option for pre-auth calls (login, register)

### Endpoint Categories (from API Endpoint Map)

**1. Auth (fully wired)**
- `POST /auth/login`, `/auth/register`, `/auth/validate-email`, `/auth/verify-otp`, `/auth/resend-otp`
- `GET /auth/google` (popup flow), `POST /auth/forgot-password`, `/auth/reset-password`
- `POST /auth/refresh` (cookie-based)

**2. Workspace (partially wired)**
- `GET /workspaces` (validated), `POST /workspaces`, `GET/PUT /workspaces/:id`
- `GET /workspaces/:id/members`, `PUT /workspaces/:id/members/:userId/role`, `DELETE /workspaces/:id/members/:userId`
- `POST /invitations`, `POST /invitations/:id/resend`, `DELETE /invitations/:id`

**3. Folders/Projects (fully wired)**
- `GET /projects`, `POST /projects`, `PUT/DELETE /projects/:id`
- `GET /datasets?projectId=`, `POST /:projectId/lists/empty`, `POST /datasets` (CSV), `DELETE/PUT /lists/:listId`, `POST /lists/:listId/duplicate`

**4. List/Grid Data (fully wired)**
- `GET /lists/:listId/data` (paginated), `PUT /lists/:listId/data` (edit/paste), `POST /lists/:listId/data` (add row), `DELETE /lists/:listId/data` (delete rows)
- `POST /lists/:listId/add-rows` (bulk add)
- Column configs: `POST/PUT/DELETE /column-configs/:id`, change type, validate conversion
- Column types: `text | email | phone | url | number | boolean | date | paragraph`

**5. Enrichment (wired)**
- `POST /lists/:listId/enrichment/estimate` (pre-flight credit check)
- `POST /lists/:listId/enrich` (start job)
- `GET /lists/:listId/enrichment/jobs`, `GET /enrichment/jobs/active`
- `GET/DELETE /enrichment/jobs/:jobId`, `POST /enrichment/jobs/:jobId/cancel`

**6. Exports/Channel Sync (8 flows wired)**
- LinkedIn Ads: accounts, ad accounts, upload-audience, OAuth
- Google Sheets: accounts, OAuth, spreadsheets, sheet tabs, export
- Meta Ads: accounts, OAuth, ad accounts, create-audience, upload-audience-data
- Reddit Ads: accounts, OAuth, businesses, ad accounts, create/upload audience
- HubSpot: accounts, OAuth, properties, create property, export-mapped
- SmartLeads: connected accounts, validate-key, connect, campaigns, sync-leads
- Mailshake: connected accounts, validate-key, connect, campaigns, create-campaign, sync-recipients
- Google Ads: accounts, OAuth, customers, create/upload audience

**7. Live Audiences (wired)**
- Account selection (LinkedIn, Meta, SmartLead)
- Audience uploads, sync, LinkedIn/Meta campaign attachment

**8. Integrations (wired)**
- CRUD: list/disconnect/edit connected accounts
- OAuth flows: 6 providers (LinkedIn, Meta, Reddit, HubSpot, Google Sheets, Google Ads) with postMessage + polling fallback
- API key flows: 4 providers (SmartLeads, Apollo, Klaviyo, Mailshake) with validate-then-connect
- Airbyte setup for LinkedIn and Meta

**9. Payments/Billing (wired, path fixes applied)**
- `GET /payments/subscription`, `POST /payments/checkout-session`
- `POST /payments/create-portal-session`, `/payments/enable-overage`
- `POST /payments/preview-upgrade`, `/payments/upgrade-subscription`, `/payments/buy-credits`
- `GET /payments/transactions-enriched`, `/payments/credits-breakdown`

**10. User Profile/API Keys**
- `PUT /users/profile`, `GET/POST /keys`, `POST /keys/:id/revoke`, `DELETE /keys/:id`
- Team keys: `GET /keys/workspace/:id/members`, `GET /keys/workspace/:id`, admin generate/revoke/delete

**11. Template System**
- `POST /lists/:listId/auto-map-template` (AI mapping + column creation)
- `GET /lists/:listId/template-mappings`, `GET /template-mappings`
- Template CRUD on mappings: add/edit/delete columns, formula columns
- External: `POST https://n8n.icustomer.ai/webhook/field-mapping-agent` (AI field matching)
- `POST /:projectId/lists/sample-data` (zero-credit demo list)

**12. Partner/Promo (wired)**
- `POST /promo/generate-batch`, `GET /promo/my-coupons`, `GET /promo/stats`, `DELETE /promo/revoke/:id`

### Socket Events

**Grid events (underscore format):**
- `row_enriched`, `enrichment_progress`
- `row_enrichment_queued`, `row_enrichment_processing`, `row_enrichment_completed`

**Notification events (colon format):**
- `enrichment:started`, `enrichment:progress`, `enrichment:completed`, `enrichment:failed`, `enrichment:cancelled`
- `row:enrichment:status`

**Template/validation events:**
- `job:update`, `job:broadcast`, `validation:update/progress/completed`
- `normalization:progress/completed`, `row_validation_updated`, `webhook:data_added`

### Missing/Requested Endpoints (from API endpoint map)
1. `GET /plans` or `GET /pricing/plans` - plan prices hardcoded
2. `GET /templates/registry` - template activation/access control
3. `GET /workspaces/:id/credits` - per-workspace credit allocation
4. `GET /workspaces/:id/roles` - dynamic role definitions
5. `GET /features/{key}/check` - feature flag check (pass-through stub exists)
6. `POST /workspaces/:id/leave` - leave workspace
7. `X-Workspace-Id` header middleware - workspace authorization boundary (CRITICAL security gap)
8. `PATCH /workspaces/:id/default` - set default workspace server-side
