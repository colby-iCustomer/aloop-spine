## Regression Flags
- Old repo used **Jotai** for state management. New repo uses React Context + React Query. Migration is documented as complete.
- Old repo used **AG Grid v32 Enterprise**. New repo uses v35 with JS theming API.
- Old repo used **Axios** (`AuthClient`) alongside fetch-based `ApiClient`. New repo unified on single fetch-based client.
- Old repo used **Ant Design (antd v6)** alongside shadcn. New repo removed antd entirely.
- Old repo had **highlight.run** for error tracking. Not documented as implemented in new repo.
- Old repo had **PostHog** for analytics. Not documented as implemented in new repo.
- Old repo had **Intercom** for support widget. Not documented as implemented in new repo.
- Old repo had **TipTap** for rich text editing (column reference autocomplete). Not mentioned in new repo docs.
- Old repo had **dnd-kit** for drag and drop. Not mentioned in new repo docs.
- Old repo had **Recharts** for charts. Not mentioned in new repo docs.
- Old repo had **ReactFlow** for flow diagrams. Not mentioned in new repo docs.
- Old repo had **react-querybuilder** for conditional enrichment filter rules. Not mentioned in new repo docs.
- Old repo had **three.js / @react-three** for 3D visuals. Documented as "not in scope" for new repo.
